# lumgr

Linux User Manager — a simple web panel to manage Linux host users.

Goal: run a containerized Web UI (HTTP port `14392`) to manage a Linux host’s users and groups.

Status: scaffold/framework only. User-management and authentication are not implemented yet.

- Authentication (JWT, password login): see `docs/auth.md`
- Running (Docker Compose): see `docs/run.md`
